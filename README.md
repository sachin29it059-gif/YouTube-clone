# 🎥 YouTube Homepage Clone

A responsive front-end clone of the YouTube homepage built using HTML5 and CSS3. The project recreates YouTube's desktop interface with a responsive layout, sidebar navigation, and video grid.



## 📂 GitHub Repository
https://github.com/yourusername/youtube-homepage-clone

## ✨ Features
- Responsive desktop layout
- YouTube-inspired header
- Sidebar navigation
- Responsive video grid
- Semantic HTML5
- CSS Grid & Flexbox
- SVG icons
- Hover effects

## 🛠️ Tech Stack
- HTML5
- CSS3
- Flexbox
- CSS Grid
